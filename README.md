Employee Management Dashboard

A modern, responsive Employee Management Dashboard built with React.js and Tailwind CSS.
This application demonstrates real-world frontend architecture, authentication handling, protected routing, performance optimization, and responsive UI design.

Fallow the step to run this project ->
1. install node moduel - npm install
2. run the project - npm run dev
3. login - add email & password
4. employee dashboard - redirect to employee dashboard
5. perform crud operations.
   
Project Overview
The Employee Management Dashboard allows organizations to manage employees efficiently with features like:
Secure login & protected routes
Employee listing with active/inactive status
Employee status toggle (activate/deactivate)
Attendance, notifications, and reports modules
Fully responsive UI for mobile, tablet, and desktop
Clean, scalable, and maintainable code structure

The project follows industry-standard React practices including lazy loading, reusable components, and optimized rendering.
Tech Stack Used
Frontend
React.js (Functional Components & Hooks)
React Router DOM (HashRouter)
Tailwind CSS (Utility-first styling)
JavaScript (ES6+)
State & Performance
React Hooks (useState, useMemo, useCallback)
Lazy Loading with React.lazy & Suspense

Other
LocalStorage (temporary data persistence)
SVG Icons

Responsive & mobile-first design
